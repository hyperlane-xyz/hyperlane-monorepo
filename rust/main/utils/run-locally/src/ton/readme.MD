You can set the variables directly in your shell:

```shell

export WALLET_VERSION="v4"
export API_KEY="your-api-key"
export DISPATCH_DOMAIN="777001"
export TARGET_DOMAIN="777002"
export TON_ADDRESS="EQC2..."
```
