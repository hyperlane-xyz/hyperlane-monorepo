# Warp Widget Examples

## Vanilla iframe (`vanilla/`)

A single HTML file — no build tools, no dependencies. Just open `index.html` in a browser.

## React App (`react-app/`)

A Vite + React app showing both integration methods:

- **Component** — `<HyperlaneWarpWidget />` from `@hyperlane-xyz/warp-widget/react`
- **Imperative** — `createWarpWidget()` from `@hyperlane-xyz/warp-widget`

```bash
cd react-app
pnpm install
pnpm dev
```
